# Geometry Dash Clone 🕹️

A 2D platformer inspired by Geometry Dash, made using Unity.

## 🚀 Features
- Endless side-scrolling mechanics
- Jump, obstacles, and game over system
- Unity C# scripting

## 📁 How to Run
1. Clone this repo
2. Open in Unity 2022+
3. Press Play ▶️

## 🧠 Built With
- Unity & C#
